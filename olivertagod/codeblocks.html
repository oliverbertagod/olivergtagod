How to install Code::blocks

About

Code::Blocks is a free, open-source cross-platform IDE that supports multiple compilers including
GCC, Clang and Visual C++. It is developed in C++ using wxWidgets as the GUI toolkit. Using a plugin architecture,
its capabilities and features are defined by the provided plugins.




Installation:

Part 1:

1. Go to Code::blocks's official website: click here
2. Then, Click " DOWNLOAD ".

Part 2:

1.Open your terminal or simple press " CTRL + ALT + T" keys.

2. Then, type or paste this commands on your terminal one by one:



$ sudo add-apt-repository ppa:codeblocks-devs/release

$ sudo apt update

$ sudo apt install codeblocks codeblocks-contrib



3. Once the download complete, open application menu then search vs code then Launch. Enjoy Coding.
