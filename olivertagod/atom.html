How to install Atom

About

Atom is a free and open-source text and source code editor for macOS, Linux, and Microsoft Windows
with support for plug-ins written in Javascript, and embedded Git Control, developed by GitHub.
Atom is a desktop application built using web technologies.




Installation:

Part 1:

1. Go to Atom's official website: click here
2. Then, Click " DOWNLOAD ".

Part 2:

1.Open your terminal or simple press " CTRL + ALT + T" keys.

2. Then, type or paste this commands on your terminal one by one:



$ sudo apt update

$ sudo apt install software-properties-common apt-transport-https wget

$ wget -q https://packagecloud.io/AtomEditor/atom/gpgkey -O- | sudo apt-key add -

$ sudo add-apt-repository "deb [arch=amd64] https://packagecloud.io/AtomEditor/atom/any/ any main"

$ sudo apt install atom



3. Once the download complete, open application menu then search Atom then Launch. Enjoy Coding.
