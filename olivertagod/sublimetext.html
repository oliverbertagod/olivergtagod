How to install Code::blocks

About

Sublime Text editor is a sophisticated text editor which is widely used among developers.
It includes wide features such as Syntax Highlight, Auto Indentation, File Type Recognition,
Sidebar, Macros, Plug-in and Packages that make it easy for working with code base.
This tutorial gives you a comprehensive coverage of concepts of Sublime Text and makes you
comfortable to use it in your software development projects.




Installation:

Part 1:

1. Go to Sublime Text's official website: click here
2. Then, Click your OS version to download the executable file.

Part 2:

1.Open your terminal or simple press " CTRL + ALT + T" keys.

2. Then, type or paste this commands on your terminal one by one:



$ wget -qO - https://download.sublimetext.com/sublimehq-pub.gpg | sudo apt-key add -

$ sudo apt-get install apt-transport-https

$ echo "deb https://download.sublimetext.com/ apt/stable/" | sudo tee /etc/apt/sources.list.d/sublime-text.list

$ sudo apt-get update

$ sudo apt-get install sublime-text



3. Once the download complete, open application menu then search sublime text then Launch. Enjoy Coding.
